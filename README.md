# Ages Mining — Cave-In Mod for NeoForge 1.21.1

Реалистичные обвалы шахт, вдохновлённые TerraFirmaCraft. Копай осторожно!

---

## Механика обвалов

### Как работают обвалы

1. **Разрушаешь блок** под землёй (ниже Y настроенного порога, по умолчанию 55).
2. **Движок стабильности** проверяет все блоки потолка в радиусе вокруг сломанного.
3. Потолок держится **только на шахтных опорах** (стойки/балки), как в TFC.
4. **С вероятностью** `baseCollapseChance` (усиленной для слабых блоков вроде песка/гравия) нестабильный блок помечается для обвала.
5. После задержки `collapseDelayTicks` блок падает как FallingBlockEntity (Minecraft-физика).
6. **Цепная реакция**: каждый упавший блок проверяет соседей (радиус `collapsePropagationRadius`).

### Предупреждения

- **Частицы пыли** падают с нестабильного потолка за `collapseDelayTicks` тиков до обвала.
- **Звуки** — треск и гул перед обвалом.
- **Action bar** предупреждает игрока во время копания под нестабильным потолком.

## Опоры шахты

### Опора-столб (Mine Support Pillar)
```
Крафт:  [ ]  [L]  [ ]
        [ ]  [L]  [ ]    →  3 столба
        [ ]  [L]  [ ]

L = любое бревно
```
- Ставится **вертикально** от пола до потолка.
- Поддерживает все блоки потолка в радиусе `supportPillarRange` (по умолчанию 5) вертикально.
- Блоки соединяются и образуют цельную опору.

### Опора-балка (Mine Support Beam)
```
Крафт:  [L]  [L]  [L]
        [S]  [ ]  [S]    →  4 балки

L = любое бревно, S = палка
```
- Ставится **горизонтально** вдоль потолка.
- Поддерживает потолок на расстоянии до `supportBeamRange` (по умолчанию 7) блоков в каждую сторону.
- Ориентируется автоматически по направлению взгляда игрока.

### Стратегия расстановки опор

```
Оптимальная сетка (опоры через 4 блока):

  P===B===B===B===P===B===B===B===P
  |               |               |
  |   Безопасная  |   Безопасная  |
  |     зона      |     зона      |
  |               |               |
  P===B===B===B===P===B===B===B===P

P = Столб, B = Балка
```

## Настройки (`agesmining-common.toml`)

### [cave_ins]
| Параметр | По умолчанию | Описание |
|---|---|---|
| `enabled` | `true` | Включить механику обвалов |
| `minDepthForCollapse` | `55` | Минимальный Y для обвалов |
| `baseCollapseChance` | `0.35` | Базовая вероятность обвала (0–1) |
| `checkRadius` | `6` | Радиус проверки вокруг сломанного блока |
| `supportPillarRange` | `5` | Вертикальный радиус столба |
| `supportBeamRange` | `7` | Горизонтальный радиус балки |
| `collapsePropagationRadius` | `3` | Радиус цепной реакции |
| `maxBlocksPerCollapse` | `64` | Макс. блоков за один обвал |
| `collapseDelayTicks` | `40` | Задержка перед обвалом (20 тиков = 1 сек) |

### [performance]
| Параметр | По умолчанию | Описание |
|---|---|---|
| `stabilityCheckInterval` | `100` | Тиков между фоновыми проверками (0 = выкл) |
| `useAsyncChecks` | `true` | Асинхронные проверки (рекомендуется для серверов) |
| `maxChecksPerTick` | `8` | Макс. проверок за тик |
| `chunkCacheSize` | `256` | Размер кэша стабильности чанков |

### [effects]
| Параметр | По умолчанию | Описание |
|---|---|---|
| `warningParticles` | `true` | Частицы-предупреждения перед обвалом |
| `enableSounds` | `true` | Звуки обвала |
| `screenShake` | `true` | Тряска экрана |

### [gameplay]
| Параметр | По умолчанию | Описание |
|---|---|---|
| `damagePlayers` | `true` | Обвал наносит урон игрокам |
| `damagePerBlock` | `1.5` | Урон за каждый упавший блок |
| `destroyItems` | `false` | Уничтожать ли предметы при обвале |

### [stability] — стабильность блоков
| Блок | Значение | Примечание |
|---|---|---|
| Камень / булыжник | `10` | Стандартная стабильность |
| Глубинный сланец | `14` | Очень прочный |
| Руда | `8` | Ослаблена пустотами |
| Земля / грунт | `5` | Средняя стабильность |
| Гравий | `3` | Очень нестабилен |
| Песок | `2` | Крайне нестабилен |

## Теги блоков (для датапаков/других модов)

| Тег | Описание |
|---|---|
| `agesmining:non_collapsible` | Блоки, которые никогда не обрушатся |
| `agesmining:non_structural` | Блоки с нулевой структурной поддержкой |
| `agesmining:mine_supports` | Все опорные блоки |
| `agesmining:all_ores` | Все рудные блоки |

## Производительность

Мод оптимизирован для серверов:

- **Очередь проверок** — обработка не более `maxChecksPerTick` за тик.
- **LRU-кэш** — повторные вычисления для уже проверенных зон кэшируются.
- **Async-режим** — проверки вынесены из основного тика.
- **Без поиска опоры по стенам** — учитываются только опоры игрока (стойки/балки).
- **Фоновые проверки** — периодический sweep только вокруг игроков.

## Установка

1. Установи **NeoForge 21.1.0+ (Minecraft 1.21.1)**
2. Положи `agesmining-1.1.0.jar` в папку `mods/`
3. Запусти игру — конфиг появится в `config/agesmining-common.toml`

## Совместимость

- Работает с любыми модами, добавляющими блоки.
- Для исключения блоков от обвала — добавь их в тег `agesmining:non_collapsible` через датапак.
- Протестировано с: Create, Quark, Biomes O' Plenty.
