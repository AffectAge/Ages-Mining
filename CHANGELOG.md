# Ages Mining Changelog

## [1.1.0] — 2026-04-22

### Изменено
- Порт сборки с **Minecraft Forge 1.20.1** на **NeoForge 1.21.1**.
- Обновлена Gradle-конфигурация на `net.neoforged.moddev`.
- Метафайл мода переименован в `neoforge.mods.toml` и обновлены диапазоны версий зависимостей.
- Обновлён `pack_format` для ресурсов Minecraft 1.21.1.
- README обновлён под новый лоадер и версию игры.
- Код мода перенесён на NeoForge API: обновлены импорты событий/регистров, конфиг-спека и идентификаторы ResourceLocation для 1.21.1.

## [1.0.0] — 2024

### Добавлено
- **Механика обвалов** — блоки потолка без опоры обрушаются через некоторое время
- **BFS-алгоритм стабильности** с ограничением глубины по `maxSupportedDistance`
- **Блок «Опора-столб»** (Mine Support Pillar) — вертикальная опора с фларенными торцами
- **Блок «Опора-балка»** (Mine Support Beam) — горизонтальная балка для потолка
- **Цепные обвалы** — обрушение соседних нестабильных блоков (радиус `collapsePropagationRadius`)
- **Предупреждающие частицы** — пыль падает с нестабильного потолка перед обвалом
- **Тряска экрана** — клиентский эффект при обвале вблизи игрока
- **Подсветка опор** — зелёные контуры при держании опоры в руке
- **Action bar предупреждения** — сообщение при копании под нестабильным потолком
- **Команды** (`/agesmining status|check|collapse|toggle|debug|reload`)
- **Вкладка в Creative Menu** с обоими блоками
- **Тултипы** с текущими значениями радиуса из конфига
- **Сетевой пакет** screen shake (сервер → клиент)
- **30+ параметров конфигурации** в `agesmining-common.toml`
- **Теги блоков**: `non_collapsible`, `non_structural`, `mine_supports`, `all_ores`
- **Рецепты крафта**: столб (3 бревна → 3 шт.), балка (3 бревна + 2 палки → 4 шт.)
- **Поддержка локализации**: `en_us` + `ru_ru`
- **Датагенерация**: blockstates, item models, recipes, tags, loot tables
- **Оптимизация**: очередь проверок, кэш чанков, rate-limiting, async-режим
